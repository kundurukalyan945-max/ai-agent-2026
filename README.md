# AI Agent Suite 2026 — Built by Kalyan

A collection of 5 AI agents built with Python, Groq API, and LangChain.
Built for: Getting an AI/ML job in Bengaluru in 2026.

## Agents Built
- my_first_ai.py — AI chatbot using Groq API
- ai_coach.py — Personal career coach with memory
- job_scraper.py — Scrapes AI/ML jobs in Bengaluru automatically
- resume_tailor.py — Tailors resume to any job description using LLM
- interview_prep.py — AI interviewer that scores your answers

## Tech Stack
Python 3.12 · Groq API · LLaMA 3.3 · JobSpy · Pandas

## Run any agent
python ai_coach.py
